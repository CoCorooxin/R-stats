5.Hypothse

A partir de lobservation sur les donnes, en particulier le rsum statistique de donnes par groupes despces, on peut formuler lhypothse tel que les variations morphologiques des fleurs diris de trois espces peuvent tre quantifi, savoir que les trois espces peuvent tre discrimin avec des mthodes statistiques. (Si on a les msures sur les ptales dune fleur diris, on peut prdire leurs espce avec des mthodes discriminatoires)

i.Premirement pour tester lhypothse, on va construire un modle linaire partir dun variable dpendant(La longueur de ptales) et le variable indpendant(espces).

mdl.lm <- iris %>% 
  lm( Petal.Length ~ Species, data = .)

kable(summary(mdl.lm)$coef, digits = 3)

A partir du modle linar obtenu, on peut concluer quil y a bien une diffrence importance de la longueur de ptales entre les spces. Cependant, est-ce que cette conclusion peut tre gnralis pour la largeur de ptales?

En fait, on peut facilement visualiser la rponse cette question avec les plots linaires.

6.La visualisation

a.Petal.Length vs Petal.Width

## `geom_smooth()` using formula 'y ~ x'

iris %>% 
  ggplot(aes(x = Sepal.Length,
             y = Sepal.Width,
             colour = Species)) +
  theme_bw() +
  geom_point()+
  geom_smooth(method = "lm")+
  labs(x = "Sepal.Length", y = "Sepal.Width", title = "Sepal.Length vs Sepal.Width", subtitle = "with a trend line") + # add labels
  theme(text = element_text(size = 15)) + # increase size of plot
  theme(legend.position = "bottom", legend.title = element_blank())  # remove legend title and change position
## `geom_smooth()` using formula 'y ~ x'

1.Les deux plots(a, b) montre que lhypothse est prouv vrai.Cest dire que si on a les msures sur les ptales dune fleur diris, on peut prdire leurs espce avec des mthodes discriminatoires (ses msures de ptales tombes dans quel groupe despces)

2. partir de plot a, on voit que la discrimination entre les trois espces est la plus important lors quon prend en compte les msures de ptals(Length et Width)

3. partir de plot b, on voit que la diffrence entre les espces est plutt flou lors quon compare les spales.
